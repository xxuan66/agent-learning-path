# 06-学习资源

这一部分不是让你把资料全部看完，而是帮助你在不同阶段找到更合适的补充材料。

建议把这里当作“按需查阅的资源导航”，而不是主学习入口。

## 本章内容

- [在线课程](在线课程.md) - 用来补体系和系统性认知
- [技术博客](技术博客.md) - 用来跟进高质量实践和观点
- [开源项目](开源项目.md) - 用来观察真实实现和工程取舍

## 怎么使用这部分

1. 缺框架认知时，优先回到仓库主线章节
2. 缺实践样例时，再看开源项目
3. 缺视野和趋势判断时，再看博客和课程
