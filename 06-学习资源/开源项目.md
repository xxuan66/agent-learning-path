# 开源项目

这一页的重点不是追 star 数，而是帮助你判断：一个项目值得学什么、适合在哪个阶段看、它在 Agent 生态里扮演什么角色。

## 🌟 先看项目时该关注什么

建议优先看这几件事：

1. 项目的核心定位
2. 它解决的是单 Agent、工作流还是多 Agent 问题
3. 它更偏教学、原型验证，还是工程化落地
4. 文档、社区和更新节奏是否健康

## 🌟 必学项目

### 1. LangChain
- **GitHub**: https://github.com/langchain-ai/langchain
- **Stars**: 80k+
- **学习点**: 框架设计、工具集成

### 2. AutoGen
- **GitHub**: https://github.com/microsoft/autogen
- **Stars**: 30k+
- **学习点**: 多 Agent 协作

### 3. CrewAI
- **GitHub**: https://github.com/joaomdmoura/crewai
- **Stars**: 15k+
- **学习点**: 角色驱动设计

### 4. Chroma
- **GitHub**: https://github.com/chroma-core/chroma
- **Stars**: 15k+
- **学习点**: 向量数据库

### 5. LlamaIndex
- **GitHub**: https://github.com/run-llama/llama_index
- **Stars**: 25k+
- **学习点**: 数据索引和检索

## 🔧 实用工具

### 1. FastChat
- **用途**: LLM 服务和评测
- **链接**: https://github.com/lm-sys/FastChat

### 2. Ollama
- **用途**: 本地运行 LLM
- **链接**: https://github.com/ollama/ollama

### 3. AnythingLLM
- **用途**: 私有知识库
- **链接**: https://github.com/Mintplex-Labs/anything-llm

## 📚 学习项目

### 入门级
- **BabyAGI** - 简单清晰
- **AutoGPT** - 自主 Agent

### 进阶级
- **ChatDev** - 软件开发团队
- **MetaGPT** - 多角色协作

## 一个更实用的学习顺序

- 想理解框架生态：先看 LangChain、AutoGen、CrewAI
- 想理解知识检索：看 Chroma、LlamaIndex
- 想理解本地部署：看 Ollama、FastChat
- 想理解多 Agent 协作：看 ChatDev、MetaGPT

## 🎯 参与贡献

1. Fork 感兴趣的项目
2. 阅读贡献指南
3. 从简单的 issue 开始
4. 提交 PR
