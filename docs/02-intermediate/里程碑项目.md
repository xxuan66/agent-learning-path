## 🏆 开创性项目

### AutoGPT (2023)

**GitHub**: https://github.com/Significant-Gravitas/Auto-GPT

**核心创新**：
- 自主目标追求
- 任务自动分解
- 记忆管理
- 文件操作能力

**架构**：
```
用户设定目标 → AutoGPT 自主规划 → 执行子任务 → 评估结果 → 调整策略
                    ↑                                        │
                    └──────────── 循环迭代 ←─────────────────┘
```

**示例**：
```python
# 目标：研究 AI 发展趋势并生成报告
# AutoGPT 自动执行：
1. 搜索最新的 AI 研究论文
2. 阅读并提取关键信息
3. 整理和分类信息
4. 撰写研究报告
5. 保存为 PDF 文件
```

**影响**：展示了 LLM Agent 的自主能力，引发广泛关注

### BabyAGI (2023)

**GitHub**: https://github.com/yoheinakajima/babyagi

**核心创新**：
- 任务优先级管理
- 简洁的架构设计
- 易于理解和扩展

**工作流程**：
```
1. 从待办列表取出最重要任务
2. 执行任务
3. 根据结果创建新任务
4. 重新排序待办列表
5. 重复
```

**代码示例**：
```python
def babyagi_cycle(objective):
    task_list = create_initial_tasks(objective)
    
    while task_list:
        # 获取最高优先级任务
        task = get_highest_priority_task(task_list)
        
        # 执行任务
        result = execute_task(task)
        
        # 基于结果创建新任务
        new_tasks = create_new_tasks(task, result, objective)
        
        # 更新任务列表
        task_list = update_task_list(task_list, new_tasks)
```

**影响**：简化了 Agent 设计思路，被广泛学习和模仿

### LangChain (2022-)

**GitHub**: https://github.com/langchain-ai/langchain

**核心贡献**：
- 标准化的 Agent 接口
- 丰富的工具集成
- 记忆管理组件
- 链式调用机制

**核心概念**：
```python
from langchain.agents import initialize_agent, Tool
from langchain.llms import OpenAI

# 定义工具
tools = [
    Tool(
        name="Search",
        func=search_engine,
        description="搜索互联网"
    ),
    Tool(
        name="Calculator",
        func=calculator,
        description="执行数学计算"
    )
]

# 初始化 Agent
agent = initialize_agent(
    tools,
    llm=OpenAI(),
    agent="zero-shot-react-description"
)

# 使用 Agent
agent.run("今天北京天气如何？温度乘以 2 是多少？")
```

**生态系统**：
- LangChain：基础框架
- LangGraph：复杂工作流
- LangSmith：调试和监控

**影响**：成为最流行的 Agent 开发框架

### AutoGen (2023)

**GitHub**: https://github.com/microsoft/autogen

**核心创新**：
- 多 Agent 对话协作
- 灵活的对话模式
- 人类参与机制

**对话模式**：
```
1. 单 Agent 模式：一个 Agent 完成任务
2. 双 Agent 模式：两个 Agent 对话协作
3. 群聊模式：多个 Agent 参与讨论
4. 人机协作：人类和 Agent 共同参与
```

**示例**：
```python
from autogen import ConversableAgent

# 创建 Agent
coder = ConversableAgent(
    name="Coder",
    system_message="你是程序员，负责写代码"
)

reviewer = ConversableAgent(
    name="Reviewer",
    system_message="你是代码审查员，负责检查代码质量"
)

# 启动对话
coder.initiate_chat(
    reviewer,
    message="请帮我审查这段代码..."
)
```

**影响**：开创了多 Agent 协作的新范式

### CrewAI (2023-)

**GitHub**: https://github.com/joaomdmoura/crewai

**核心特点**：
- 角色定义清晰
- 任务流程化
- 团队协作模拟

**示例**：
```python
from crewai import Agent, Task, Crew

# 定义角色
researcher = Agent(
    role='高级研究员',
    goal='深入调研主题',
    backstory='你是一位经验丰富的研究员...'
)

writer = Agent(
    role='内容作家',
    goal='撰写高质量文章',
    backstory='你是一位优秀的作家...'
)

# 定义任务
research_task = Task(
    description='调研 AI 发展趋势',
    agent=researcher
)

write_task = Task(
    description='基于调研结果撰写文章',
    agent=writer
)

# 创建团队
crew = Crew(
    agents=[researcher, writer],
    tasks=[research_task, write_task],
    verbose=True
)

# 执行
result = crew.kickoff()
```

**影响**：使多 Agent 协作更加结构化和易用

## 🔬 研究项目

### ReAct (2022)

**论文**: [ReAct: Synergizing Reasoning and Acting](https://arxiv.org/abs/2210.03629)

**核心思想**：推理（Reasoning）和行动（Acting）协同

**模式**：
```
Thought: 我需要先查询相关信息
Action: Search[AI Agent 发展趋势]
Observation: 根据搜索结果...
Thought: 现在我有了足够信息
Action: Finish[生成最终答案]
```

**影响**：成为 Agent 推理的标准范式

---

### Tree of Thoughts (2023)

**论文**: [Tree of Thoughts: Deliberate Problem Solving](https://arxiv.org/abs/2305.10601)

**核心创新**：
- 多路径探索
- 前瞻和回溯
- 自我评估

**搜索策略**：
```
        初始问题
       /   |   \
     思路 1 思路 2 思路 3
    /  \   /  \   /  \
   ...  ... ... ... ... ...
   
BFS: 广度优先搜索
DFS: 深度优先搜索
```

**影响**：提升了复杂问题的解决能力

## 🛠️ 基础设施项目

### Chroma (2022-)

**GitHub**: https://github.com/chroma-core/chroma

**用途**：向量数据库，用于 Agent 记忆

**特点**：
- 简单易用
- 本地优先
- 支持嵌入存储

**示例**：
```python
from chromadb import Client

client = Client()
collection = client.create_collection("agent_memory")

# 添加记忆
collection.add(
    documents=["用户喜欢 Python", "项目截止日期是周五"],
    ids=["mem1", "mem2"]
)

# 检索记忆
results = collection.query(
    query_texts=["用户偏好"],
    n_results=2
)
```
