# 开源项目

## 🌟 必学项目

### 1. LangChain
- **GitHub**: https://github.com/langchain-ai/langchain
- **Stars**: 80k+
- **学习点**: 框架设计、工具集成

### 2. AutoGen
- **GitHub**: https://github.com/microsoft/autogen
- **Stars**: 30k+
- **学习点**: 多 Agent 协作

### 3. CrewAI
- **GitHub**: https://github.com/joaomdmoura/crewai
- **Stars**: 15k+
- **学习点**: 角色驱动设计

### 4. Chroma
- **GitHub**: https://github.com/chroma-core/chroma
- **Stars**: 15k+
- **学习点**: 向量数据库

### 5. LlamaIndex
- **GitHub**: https://github.com/run-llama/llama_index
- **Stars**: 25k+
- **学习点**: 数据索引和检索

### 1. FastChat

### 2. Ollama
- **用途**: 本地运行 LLM
- **链接**: https://github.com/ollama/ollama

### 3. AnythingLLM

### 5. OpenClaw
- **用途**: 多渠道 AI Agent 框架，技能市场生态
- **链接**: https://github.com/openclaw/openclaw

## 📚 学习项目

### 入门级
- **BabyAGI** - 简单清晰
- **AutoGPT** - 自主 Agent

### 进阶级
- **ChatDev** - 软件开发团队
- **MetaGPT** - 多角色协作

### 2025-2026 新星
- **OpenAI Swarm** - 轻量级多 Agent 编排
- **Browser Use** - Agent 浏览器自动化
- **MCP (Model Context Protocol)** - Agent 工具连接标准
- **LangGraph** - Agent 工作流图编排
- **OpenClaw** - 多渠道 AI Agent 框架
