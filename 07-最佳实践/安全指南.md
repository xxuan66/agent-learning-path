# 安全指南

安全问题在 Agent 系统里往往不是补丁，而是基础约束。因为一旦模型能调用工具、访问资源、连接外部系统，风险就不再只是“说错话”，而可能变成真实执行问题。

## ⚠️ 安全风险

### 1. 提示注入

```python
# ❌ 危险：直接拼接用户输入
prompt = f"执行以下命令：{user_input}"

# ✅ 安全：使用参数化
prompt = "执行以下命令：{command}"
formatted = prompt.format(command=user_input)
```

### 2. 工具滥用

```python
# ❌ 危险：无限制的工具调用
@tool
def execute_command(cmd: str):
    return os.system(cmd)

# ✅ 安全：白名单和验证
ALLOWED_COMMANDS = ["ls", "pwd", "cat"]

@tool
def execute_command(cmd: str):
    if cmd.split()[0] not in ALLOWED_COMMANDS:
        raise ValueError(f"命令{cmd}不被允许")
    return subprocess.run(cmd, capture_output=True, text=True)
```

### 3. 数据泄露

```python
# ❌ 危险：记录敏感信息
logger.info(f"用户查询：{query}")  # 可能包含密码等

# ✅ 安全：脱敏处理
def sanitize(text):
    text = re.sub(r'password=\S+', 'password=***', text)
    text = re.sub(r'\d{18}', 'ID:***', text)  # 身份证号
    return text

logger.info(f"用户查询：{sanitize(query)}")
```

## 一个现实判断

越是“能干活”的 Agent，越需要安全边界。不要把安全理解成上线前最后一层，而应该在设计工具、权限和日志时就一起考虑进去。

## 🔒 安全措施

### 1. 输入验证

```python
from pydantic import BaseModel, validator, Field

class SafeInput(BaseModel):
    query: str = Field(..., min_length=1, max_length=1000)
    category: str = Field(..., pattern="^(news|weather|tech)$")
    
    @validator('query')
    def validate_query(cls, v):
        # 检查注入攻击
        dangerous_patterns = ["DROP TABLE", "DELETE FROM", "<script>"]
        for pattern in dangerous_patterns:
            if pattern.lower() in v.lower():
                raise ValueError("检测到恶意输入")
        return v
```

### 2. 速率限制

```python
from functools import wraps
import time

def rate_limit(max_calls=10, period=60):
    calls = []
    
    def decorator(func):
        @wraps(func)
        def wrapper(*args, **kwargs):
            now = time.time()
            calls[:] = [t for t in calls if now - t < period]
            
            if len(calls) >= max_calls:
                raise Exception("请求过于频繁")
            
            calls.append(now)
            return func(*args, **kwargs)
        return wrapper
    return decorator

@rate_limit(max_calls=10, period=60)
def agent_query(user_input):
    return agent.invoke(user_input)
```

### 3. 输出过滤

```python
def filter_output(text: str) -> str:
    """过滤敏感输出"""
    # 移除可能的敏感信息
    patterns = [
        r'\b\d{16}\b',  # 信用卡号
        r'\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b',  # 邮箱
    ]
    
    for pattern in patterns:
        text = re.sub(pattern, '[REDACTED]', text)
    
    return text
```

## 🛡️ 权限控制

```python
class Permission:
    READ = "read"
    WRITE = "write"
    ADMIN = "admin"

def require_permission(required: Permission):
    def decorator(func):
        @wraps(func)
        def wrapper(user, *args, **kwargs):
            if user.permission_level < required:
                raise PermissionError("权限不足")
            return func(user, *args, **kwargs)
        return wrapper
    return decorator

@require_permission(Permission.WRITE)
def update_knowledge(content):
    knowledge_base.add(content)
```

## 📊 安全监控

```python
def log_security_event(event_type: str, details: dict):
    """记录安全事件"""
    logger.warning(
        f"安全事件：{event_type}",
        extra={
            "event_type": event_type,
            "details": details,
            "timestamp": time.time()
        }
    )

# 使用
log_security_event("INJECTION_ATTEMPT", {
    "user_id": user.id,
    "input": suspicious_input
})
```
