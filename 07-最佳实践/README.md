# 07-最佳实践

这一部分是把前面学到的概念真正落到工程里时，最容易决定系统质量的地方：规范、安全、性能。

很多 Agent 项目之所以难以落地，不是因为模型不够强，而是因为这三件事没有处理好。

## 本章内容

- [开发规范](开发规范.md) - 代码质量和工程实践
- [安全指南](安全指南.md) - 安全防护和风险控制
- [性能优化](性能优化.md) - 提升效率和降低成本

## 怎么读这一部分

1. 先看安全，再看性能
2. 先建立边界，再做优化
3. 把“能跑”升级成“能长期运行”
