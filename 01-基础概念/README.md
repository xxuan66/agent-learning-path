# 01-基础概念

这一部分解决的是最基础但也最容易混淆的问题：到底什么才算 Agent，它和普通聊天助手、工作流、自动化脚本的边界在哪里。

## 本章内容

- [什么是 AI Agent](什么是 AI Agent.md) - Agent 的定义与特征
- [Agent 核心能力](Agent 核心能力.md) - 规划、记忆、工具、反思
- [Agent 应用场景](Agent 应用场景.md) - 实际应用案例分析

## 学习目标

完成本章学习后，你应该能够：

1. 准确描述 AI Agent 的定义和特征
2. 理解 Agent 的核心能力和系统边界
3. 识别 Agent 在不同场景的应用价值
4. 区分 Agent、聊天助手、工具调用和传统程序
