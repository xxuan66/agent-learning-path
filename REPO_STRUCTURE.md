# 仓库整理说明

## 当前状态

本仓库已经完成从旧的 `docs/` 分层稿向顶层中文目录的主入口迁移。

当前推荐阅读入口为：

- `01-基础概念/`
- `02-技术演进/`
- `03-核心组件/`
- `04-主流框架/`
- `05-实战案例/`
- `06-学习资源/`
- `07-最佳实践/`

## docs 目录的定位

`docs/` 目录保留早期分层稿，用于：

- 历史参考
- 对照旧链接
- 后续按需整理或迁移

当前不建议再把 `docs/` 作为主入口继续扩展。

## 后续维护原则

1. 新增或更新内容时，优先更新顶层中文目录。
2. `README.md` 以顶层中文目录为主。
3. `docs/` 如无明确需要，可保持存量，不再继续扩写。
4. 如果未来要删除 `docs/`，应先确认是否仍有外部链接依赖。
