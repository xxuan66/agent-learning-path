# 03-核心组件

这一部分关注的是 Agent 真正能不能工作起来的关键模块：规划、记忆、工具使用，以及执行后的反思与优化。

如果前两部分解决的是“Agent 是什么、从哪里来”，这一部分解决的就是：“一个 Agent 为什么能完成任务，以及它通常会在哪里失效。”

## 本章内容

- [规划与任务分解](规划与任务分解.md) - 如何让 Agent 思考
- [记忆系统](记忆系统.md) - 短期记忆与长期记忆
- [工具使用](工具使用.md) - 扩展 Agent 能力
- [反思与优化](反思与优化.md) - 持续改进机制

## 学习目标

1. 理解 Agent 的关键模块分别解决什么问题
2. 分清哪些问题属于规划，哪些属于记忆，哪些属于工具边界
3. 理解为什么很多 Agent 失败并不是模型太弱，而是组件协同出了问题
4. 建立更系统的 Agent 设计视角
