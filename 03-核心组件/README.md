## 组件关系

```
用户输入
    │
    ▼
┌─────────┐
│  规划   │ ← 理解意图，拆分任务
└────┬────┘
     │
     ▼
┌─────────┐
│  记忆   │ ← 查阅历史，获取上下文
└────┬────┘
     │
     ▼
┌─────────┐
│  工具   │ ← 执行操作，获取结果
└────┬────┘
     │
     ▼
┌─────────┐
│  评估   │ ← 检查结果，决定是否继续
└────┬────┘
     │
     ▼
  用户输出
```

## 学习建议

1. **先学规划和记忆**：这两个是最基础的组件
2. **再学 Multi-Agent**：进阶内容，需要理解前两者
3. **评估和调试**：实战中非常重要，不要跳过
