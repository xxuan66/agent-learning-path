## 案例特点

### 客服助手
- 最简单的入门案例
- 演示基本的对话管理
- 包含知识库检索

### 数据分析 Agent 🆕
- 完整的数据处理流程
- 可视化图表生成
- 自动生成分析报告

### 代码生成 Agent 🆕
- 代码生成、执行、测试一体化
- 自动修复代码错误
- 代码质量审查

### 研究助手 Agent 🆕
- 多源信息搜集
- 内容理解和总结
- 自动生成研究报告

## 如何使用案例

1. **阅读案例描述**：了解场景和需求
2. **理解架构设计**：看整体结构
3. **学习工具定义**：看每个工具怎么实现
4. **运行代码示例**：亲手试一试
5. **扩展和改进**：添加自己的功能

---

## 扩展方向

每个案例都可以扩展：

- **客服助手**：添加工单系统、满意度调查
- **数据分析**：添加实时数据、预测功能
- **代码生成**：添加更多语言、测试生成
- **研究助手**：添加知识图谱、持续跟踪
