# 05-实战案例

这一部分的重点不是再做几个 demo，而是把前面学过的概念、组件和框架真正落到业务场景里。

## 本章内容

- [客服助手](客服助手.md) - 智能客服系统
- [数据分析助手](数据分析助手.md) - 自动化数据分析（规划中）
- [自动化工作流](自动化工作流.md) - 业务流程自动化（规划中）

## 学习目标

1. 理解一个 Agent 应用从需求到落地的关键环节
2. 学会在业务场景里判断哪些能力真的需要 Agent
3. 理解性能、错误处理、降级和人工接管的重要性
4. 建立更接近真实项目的设计思路
